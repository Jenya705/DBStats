mysql данные указываются в конфиге, который создатся после первого запуска плагина

либо можете указать напрямую в классе DBConfiguration данные

удалите прошлый конфиг, чтобы новый появился,
прошлая ошибка была из-за reload, возможно она сломает плагин ещё раз

если online-mode включен, то сохраняются uuid, иначе никнеймы

text_id - uuid либо никнейм
played_time - время, которое играл игрок 
hours - если 1, то время является часами иначе минутами
online - обновляется при заходе/выходе игрока

также drop table player_stats; чтобы появились новые значения в таблице

config:

timeBetweenSendingStats - время, которое плагин будет ждать для отправки статов об игроке
timeToHours - минуты, когда время превращается в часы

билд находится в папке, либо создать свой билд "mvn package"
